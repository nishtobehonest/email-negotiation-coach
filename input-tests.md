## Test 1: Hard Deadline with Implied Alternative

### Email
Dear Team,

As discussed earlier, we need confirmation by end of day Friday in order to proceed with this agreement.

If we do not hear back by then, we will move forward with another partner for this phase.

Best regards,  
Procurement Office

### Optional Context
High stakes, institutional counterparty, long-term relationship matters

### Purpose
Tests deadline as power, implied alternatives, distributive or mixed negotiation framing.

## How to Use

- Run one test at a time
- Observe consistency of categories
- Check that no replies are drafted
- Validate that checklist gaps change with context
